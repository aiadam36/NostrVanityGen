# NostrVanityGen

Nostr vanity public key generator

## How To Use

Install requirements:

```bash
pip install -r requirements.txt
```

And refer below...

### Example commands

```bash
python3 main.py -t adam -w 4 # find 'adam' with 4 background processes
```

```bash
python3 main.py --target adam --workers 4 # same as above
```
