# NostrVanityGen

Nostr vanity public key generator

## How To Use

Run:

```bash
pip install -r requirements.txt
nano main.py
```

Configure the value:

```python
TARGET = "null"  # your ideal pubkey
NUM_THREADS = 4  # amount of cpu cores allocated
```

Then run `python3 main.py`
